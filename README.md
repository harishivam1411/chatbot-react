# Chatbot React

A simple chatbot application built with React.

## Features

The chatbot responds to the following commands:

- hello
- hello chatbot
- flip a coin
- get today's date
- roll a dice

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/harishivam1411/chatbot-react.git
   ```
2. Navigate to the project directory:
   ```
   cd chatbot-react
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Start the development server:
   ```
   npm run dev
   ```
2. Open [http://localhost:5173/](http://localhost:5173) in your browser.
